# 🎨 Creative Agency Landing Page

A simple, clean **front-end landing page** for a creative agency. Built using pure HTML and CSS, focused on layout, structure, and UI design.

---

## 📌 Overview

This project is a static website that represents a creative agency. It includes multiple sections commonly found in modern landing pages like:

* Navigation bar
* Hero / intro section
* Services & features
* Portfolio showcase
* Offers
* “Why choose us” section

No frameworks. No JavaScript. Just raw front-end fundamentals done right.

---

## 🚀 Features

* Responsive layout
* Clean section-based structure
* Reusable card components
* Image overlay effects (featured work section)
* Simple and readable HTML structure
* Organized content blocks for scalability
